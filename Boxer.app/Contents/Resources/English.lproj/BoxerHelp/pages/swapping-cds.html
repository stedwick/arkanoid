Swapping CDs in multi-CD games

If your game comes on more than one CD, then it’s easy to switch to the next CD:

  • If you have a physical CD: insert it into your Mac’s DVD drive.
  • If you have a folder or disc image: drag it into the game window.

The new CD will replace the previous CD at the same drive letter. You can then tell the game to continue loading.

Swapping back and forth between CDs

Once you’ve added a new CD like this, it goes into a queue with any previous CDs. You can then swap between each CD in a queue:

  • Use the Drives ▸ Next Disc menu item to cycle through all the CDs in the queue; or
  • Go to Drives ▸ Show DOS Drives and double-click a specific CD to swap to it.

Bundling additional CDs into the gamebox

If you don’t want to fish out your CDs every time you play, you can use the Drives Inspector to import more CDs into the gamebox. Select a CD and click the icon to begin importing.

This rips the CD and makes it a part of your gamebox, which means it’s always there to swap between: just use Drives ▸ Next Disc whenever you need the next CD.
