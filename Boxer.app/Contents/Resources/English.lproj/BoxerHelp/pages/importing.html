Import a new game

Boxer installs games for you and wraps them up as gameboxes in your DOS Games folder. Here’s how:

1 Drop your game onto the Import window

Bring up the Import window by clicking on the welcome screen icon or using the File ▸ Import New Game menu item.

Then drop your game onto the window to begin. Boxer accepts:

  • CD-ROMs and floppy disks
  • Regular folders containing DOS game files
  • Disc images in .iso, .cdr, .ima, .img or .vfd format

Boxer can’t import from .bin+.cue images, but you can import them manually instead.

2 Choose (or skip) the game installer

If Boxer thinks the game needs installing, it’ll now ask you to choose which installer to run. The recommended (or only) installer is selected by default.

You can skip the installer instead, if you know the game is already installed. (This is usually the case for games downloaded from Abandonware websites.)

If you’re not sure, or you want to confirm the game’s configuration settings, then launch the installer anyway.

3 Let the game install itself

This step will be skipped if Boxer determined that the game was installed already, or if you skipped the installer yourself.

Boxer will launch the chosen installer in DOS, to let it configure the game and copy files into its new gamebox.

Follow the on-screen instructions to install the game. See game installation tips for more details.

Once the installer is done and has quit back to the DOS prompt, click “Finish importing” to continue.

4 Twiddle thumbs while Boxer finishes up

Once the DOS installer is done (or has been skipped), then Boxer will import the game’s source files into the new gamebox.

If you’re importing a CD-ROM game, then Boxer puts a copy of the CD into the gamebox: this means you won’t have to keep the original around in order to play the game.

This process may take several minutes if you’re importing from a real CD-ROM. You can skip this step if you’re impatient, and just keep the CD in the drive when you want to play.

5 Profit!

Your game is now a shiny new gamebox living in your DOS Games folder.

Boxer has guessed a suitable name for the gamebox and given it a bootleg disk icon, but you can rename it and decorate it with your own cover art if you like.

Once you’re happy with it, click “Launch game” to start playing!
