Solving common problems

Problems when importing

  • Which sound card should I choose?
  • The game’s installer says it can’t copy files, or asks for floppy disks I don’t have
  • My game comes as a .bin+.cue image and Boxer won’t import it
  • Boxer says my game is Windows-only and won’t import it
  • I have a game on a hybrid (Mac + PC) CD-ROM and Boxer won’t import it
  • My game has a patch or crack, how do I apply it?
  • My game has an expansion or add-on, how do I install it?

Problems when playing

  • My game runs too fast or too slow
  • My game has multiple CDs, how do I swap between them?

Keyboard and mouse problems

  • My game needs the numeric keypad, but my Mac keyboard doesn’t have one
  • I have a one-button mouse, but my game needs two buttons
  • The function keys don’t work
  • I keep switching OS X Spaces while trying to play
  • I can’t find the right key on my Mac keyboard

If you didn’t find an answer here, contact support with the details of the problem and I’ll help you out as best I can. Be sure to mention which game and where you got it from.
