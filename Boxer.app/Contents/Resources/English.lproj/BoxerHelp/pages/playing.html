Play your games

1 Pick a game to play

Open your DOS Games folder by clicking on the welcome screen icon or via File ▸ Browse Games Folder. This is where Boxer keeps the games you’ve imported.

Double-click a game to start playing.

2 Choose a program to run (the first time)

When you open a game for the first time, you’re greeted with a list of the game’s programs. Click a program to launch it.

The right program is usually the game’s name abbreviated: e.g. war2.exe for Warcraft II. If in doubt, try each one to see which starts up the game.

After starting a program, you can choose to always launch that program. This way the program will automatically start when you open the game.

3 Tweak game settings (if you need to)

For some games you may need to tweak the CPU speed or mouse sensitivity to get them running just right.

If so, bring up the Inspector panel by pressing Cmd ⌘+I. This lets you tweak the emulation for this game.

You can try out different graphics options in the Display Preferences too. These options apply to all games.
