Importing games from .bin+.cue disc images

Boxer can’t directly import games from .bin+.cue images. Instead, you’ll need to make your own gamebox for the image and poke around a bit in DOS:

  1. Create a new folder and put the .bin and .cue files inside it.
  2. Rename the folder to Game Name.boxer to make it into a gamebox, then doubleclick it to open it in Boxer.
  3. Switch to drive D (the location of the disc image) and run the game’s installer: usually INSTALL.EXE or similar.
  4. Tell the game to install to drive C. See also game installation tips.

Once this is complete, you’ll have an installed and playable game in its own gamebox.

This process is only necessary for .bin+.cue images: you can import games from other types of images more easily using the Import window.
